This function implements the control of arm movement. The particularity of this code is that it checks whether a waypoint has already been visited in fact, if the robot encounters this waypoint for the first time it will move the arm to either the "up" or "down" position, if on the other hand the robot has already encountered this wayoint it will have already recorded the position of the hint taken in the first visit and will change the arm position only if necessary
