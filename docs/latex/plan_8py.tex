 This script implements the plan that allows the robot to achieve its goal by performing the different actions. Until all actions are successful ( especially the \char`\"{}\+Check\+Hyp\+Action\char`\"{} action) a re-\/planning takes place. Within the script we find, for the previous purpose, several update functions.
